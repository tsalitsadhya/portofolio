Student Prediction

Developed a web-based student performance prediction system using machine learning. The system analyzes students’ activity patterns and grades to predict their likelihood of passing or failing a course. Built with Django, PostgreSQL, and a Random Forest classifier, the platform provides actionable insights through clear results and intuitive visualizations. Features include :

  • 1. Graduation prediction with confidence scores
  • 2. Learning style analysis and study partner recommendations
  • 3. Interactive dashboards for students and instructors
This project demonstrates how data-driven analytics can support students’ academic success and help educators identify at-risk students early.

Shopping Analysis

Built a market basket analysis using the Apriori algorithm and association rules to uncover customer purchasing patterns. The analysis identified frequently bought item combinations (e.g., bottled beer & sausage, frankfurter & vegetables) and provided actionable recommendations for bundle promotions and cross-selling strategies. Results include confidence scores, lift metrics, and insights to help retailers optimize product placement and marketing.

Revenue Accounting System

Developed a web-based Revenue Accounting System to record and monitor daily retail transactions. The system allows users to :

  • 1. Create and manage invoices with itemized details
  • 2. Automatically calculate total price, payment, and change
  • 3. Generate daily transaction reports, including sales recaps, revenue, and profit
  • 4. Export transaction details to PDF
Built with PHP and MySQL, this tool streamlines sales tracking and ensures accurate financial records for retail businesses.
